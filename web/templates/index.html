Project Overview

Software-defined Networking (SDN) is a novel network architecture which centralizes network intelligence by decoupling control plane from data plane. SDN makes it possible to manage networks using softwares, allowing reconfiguration in an easier and more flexible way without modifying the hardware equipments. This also enhances network performance and monitoring capacities. This project pushes forward in the scale of monitoring in SDN, aiming at capturing more information from data plane in order to realize admission control and adaptive routing which help achieve 100% link utilization under heavy load.
